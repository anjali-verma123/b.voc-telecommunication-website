To create a To-do list app using above experiments.

Mobile storage memory – 128GB.

Program
The code given below is for creating a To-do List App, which will take user’s everday tasks and make a
list of tasks. User can click on their tasks to remove them as well. The code is written in App.js. To-do
list app code is divided into 4 files – App.js which contains main code, addTodo.js which will allow the
user to enter new to-do, todoItem.js which allow user to maintain and remove elements of lists,
header.js for styling header.
